Gambit

About

Gambit is a cross-platform chess game.

Gambit currently supports the following languages:

  • English (built-in)
  • Nederlands

Downloads

Download the latest version of Gambit (for Windows users).

Users of Unix-like operating systems may want to follow the Building Gambit guide (it's really easy, guaranteed!).
